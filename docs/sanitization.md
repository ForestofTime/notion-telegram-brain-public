# Sanitization Notes

## What was removed

- Local deployment temp scripts under `/tmp/notion_*.exp`
- Local archive package used for server migration
- Local runtime project directory containing real `.env`, logs and runtime index
- Any content containing production tokens, host credentials, or private operational notes

## What is preserved

- Source code structure
- Build and run scripts
- Telegram command framework
- Notion sync and retrieval logic

## Example data policy

This repo includes `data/example-index.json` only as a schema-compatible sample.

Real workspace data should be generated by running `/sync` after configuring local secrets.

## Pre-publish checklist

1. Run `npm run security:scan`
2. Verify `.env` is not tracked
3. Verify `data/notion-index.json` is not tracked
4. Verify no host credentials in repository history
